IDM 241 - MICROINTERACTIONS | ALPHA BUILD

Descriptions:

TRIGGERS

  • user hovers on the film card
  • user hovers off the film card

RULES

  • cursor changes to pointer when mouse is hovering over the card
  • card transforms on the Y-axis 180 degrees to show a grey backside with more menu options such as liking the film, favoriting, and adding to a list.
  • cursor changes from pointer to normal while hovering off
  • card transforms on the Y-axis 180 degrees again when the user hovers off
  • 2px solid border appears around the card and disappears when hovered off

FEEDBACK

  • cursor changes to pointer when mouse is hovering over the card
  • card transforms on the Y-axis 180 degrees to show a grey backside with more menu options such as liking the film, favoriting, and adding to a list.
  • cursor changes from pointer to normal while hovering off
  • card transforms on the Y-axis 180 degrees again when the user hovers off
  • 2px solid border appears around the card and disappears when hovered off

LOOPS & MODES

  • there are no loops present
  • When the user is hovering over the image, the card flips over and reveals more menu information on how the image can be interacted with, such as liking the film, favoriting, and adding to a list. When the user hovers off, the card flips back to the film poster.
